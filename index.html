menu-breaker.js

See on GitHub

With menu-breaker.js You can switch between mobile and desktop menu on window resize.
This plugin cal also detect sub menu side based on space between side of browser window and sub menu element

Usage

At the beginning connect the library with Your project:

• using script tag in HTML:

<script src="/directory/to/library/folder/menu-breaker.js"></script>

• or via command line and CommonJS/ES6 import:

npm install menu-breaker // npm

yarn add menu-breaker // yarn
const MenuBreaker = require('menu-breaker'); // CommonJS

import MenuBreaker form 'menu-breaker'; // ES6

Next use library with:

• Vanilla JavaScript e.g:

const elem = document.querySelector('.desktop');
const menuBreaker = new MenuBreaker(elem, {
  // options...
});

• or jQuery e.g:

Connect jQuery in HTML 
<script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>

or include via command line and CommonJS 
npm install jquery // npm

yarn add jquery // yarn

bower install jquery // bower
const jQuery = require('jquery');

and call plugin on element 
$('.desktop').menuBreaker({
  // options...
});

Mobile menu

To set mobile menu, add to mobile menu container div:

data-mobile

Button to open mobile menu

To set button to open mobile menu, add to mobile menu button element:

data-open

Button to close mobile menu

To set button to close mobile menu, add to mobile menu close element:

data-close

Button to open and close mobile menu

To set button to open and close mobile menu, add to mobile menu close element:

data-open-close

Desktop menu height

To set desktop menu height, place in JavaScript (or jQuery) options:

'navbar-height': 70
