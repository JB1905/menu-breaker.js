MENU

menu-breaker.js

With menu-breaker.js You can switch between mobile and desktop menu on window resize.
This plugin cal also detect sub menu side based on space between side of browser window and sub menu element

You can use menu-breaker.js with Vanilla JavaScript e.g:

var elem = document.querySelector('.desktop');
var menuBreaker = new MenuBreaker(elem);

or jQuery:

$('.desktop').MenuBreaker();

Mobile menu

To set mobile menu, add to mobile menu container div:

data-mobile

Button to open mobile menu

To set button to open mobile menu, add to mobile menu button element:

data-open

Button to close mobile menu

To set button to close mobile menu, add to mobile menu close element
(if You want to set one button to open and close menu add this code to mobile menu open button):

data-close

Desktop menu height

To set desktop menu height, place in JavaScript (or jQuery) options:

'navbar-height': 70

Overlay options

You can style overlay element using this options:

overlay: {
	show: true,
	opacity: 0.5,
	cursor: 'pointer',
	transition: 'all 300ms ease',
	'background-color': '#000'
}
